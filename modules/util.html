referia.util.text.pdf_extract_text(filename, directory='', start_page=None, end_page=None, max_chars=None)[source]

Extract text content from a PDF file.

Uses pdfminer.six to extract text from PDF pages. Useful for feeding PDF content to LLM functions for summarization or analysis.

Parameters:

  • filename (str) The filename of the PDF file.

  • directory (str) The directory of the PDF file.

  • start_page (int, optional) Starting page number (1-indexed). If None, starts from page 1.

  • end_page (int, optional) Ending page number (1-indexed). If None, extracts to end of document.

  • max_chars (int, optional) Maximum characters to extract. If None, extracts all text.

Returns:

Extracted text content.

Return type:

str

Example:

# Extract first 5 pages
text = pdf_extract_text("thesis_ch1.pdf", directory="/path/to/pdfs",
                       start_page=1, end_page=5)

# Extract with character limit (for LLM context windows)
text = pdf_extract_text("thesis_ch1.pdf", directory="/path/to/pdfs",
                       max_chars=50000)
